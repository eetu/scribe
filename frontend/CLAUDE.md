# Frontend

Vite + React 19 + Emotion + TanStack Router (file-based) + SWR.

## Validation

```sh
yarn validate    # lint + format + typecheck in one shot
yarn dev         # vite dev server on :5173, proxies api/auth/status to :3003
yarn build       # tsc + vite build → dist/
```

Use **yarn** (not npm). Repo uses Yarn 4 via `packageManager`.

## Layout

```
src/
├── main.tsx                entry — mounts <Root> + <RouterProvider>
├── Root.tsx                ThemeProvider + global styles
├── themes.ts               ⚠ copied verbatim from ../../halo/frontend/src/themes.ts
├── mq.ts                   media-query helpers
├── api.ts                  fetch wrappers + SSE parser for job progress
├── routeTree.gen.ts        generated by @tanstack/router-plugin (stub committed)
├── routes/
│   ├── __root.tsx          layout — sidebar + outlet, wrapped in <LoginGate>
│   └── index.tsx           landing
└── components/
    ├── Wordmark.tsx        brand mark — quill stroke + accent dot, "scribe."
    └── LoginGate.tsx       401 → "sign in" button → /auth/login
```

## Conventions

- **Theme tokens.** Always `useTheme()` — no hardcoded hex outside `themes.ts`.
- **Don't fork `themes.ts`.** Copied verbatim from halo. When halo evolves,
  copy across. Only `Wordmark.tsx` is intentional divergence.
- **TanStack Router.** File-based; plugin regenerates `routeTree.gen.ts`.
- **Responsive.** Single breakpoint at 600px via `mq[0]` / `useMediaQuery`.
- **Voice.** lowercase, terse, archival/literary. Empty states allowed one
  quiet line ("no books yet. log in to begin."). No marketing fluff. No
  exclamation marks. No emoji.

## Don'ts

- Don't introduce zustand/redux. SWR + local state is enough.
- Don't add hover-only affordances — touch-first.
- Don't display Audible cover art uncached — proxy through the backend so
  ABS doesn't end up hitting Amazon CDNs while playing.
